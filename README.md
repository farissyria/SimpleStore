# Test MVC Application - ASP.NET Core E-Commerce Demo

A simple **ASP.NET Core MVC** web application for managing items/inventory with SQL Server database, following standard MVC architecture patterns.

## 🚀 Technologies Used

| Technology | Version | Purpose |
|------------|---------|---------|
| **.NET Core** | 6.0 / 7.0 / 8.0 | Framework |
| **ASP.NET Core MVC** | - | Web application framework |
| **Entity Framework Core** | Latest | ORM & Database access |
| **SQL Server** | 2019/2022 | Database |
| **Repository Pattern** | - | Data access abstraction |
| ** Views** | - | UI templating |

## 📦 NuGet Packages

| Package | Purpose |
|---------|---------|
| Microsoft.EntityFrameworkCore | Core ORM framework |
| Microsoft.EntityFrameworkCore.SqlServer | SQL Server provider |
| Microsoft.EntityFrameworkCore.Tools | Migration commands |
| Microsoft.EntityFrameworkCore.Design | Design-time tools |


## 🛠️ Features

- ✅ **MVC Architecture** - Model-View-Controller pattern
- ✅ **Repository Pattern** - Data access abstraction layer
- ✅ **Entity Framework Core** - Database operations
- ✅ **SQL Server** - Relational database
- ✅ **CRUD Operations** - Create, Read, Update, Delete
- ✅ **Dependency Injection** - Built-in IoC container
- ✅ **Generic Repository** - Reusable data access methods
 
🚀 Getting Started
Prerequisites
.NET SDK (6.0 or later)
SQL Server or SQL Server LocalDB
Visual Studio 2022 or VS Code
